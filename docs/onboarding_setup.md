# Product Onboarding and Setup Guide

## SavannahCloud Hosting
1. Sign in to the customer dashboard.
2. Open Infrastructure > Servers.
3. Choose a plan and create a server.
4. Wait for provisioning to complete.
5. Use the dashboard console or SSH to connect.
6. If needed, upgrade the plan from the server detail page.

## Agentic-Flow AI
1. Sign in to the platform dashboard.
2. Create a new workflow project.
3. Connect the required data sources or APIs.
4. Choose the model and execution rules.
5. Test the workflow before publishing.
6. Publish the workflow for production use.

## Baso Enterprise API
1. Open Developer Settings in the dashboard.
2. Generate an API key.
3. Copy the key securely and do not expose it in client code.
4. Add the Bearer token to your request headers.
5. Test your integration in a staging environment.
6. Move to production after validation.

## Getting Started Help
- If you are unsure which product fits your needs, contact the marketing team.
- Share your preferred contact details so a team member can follow up.
- Customers may also ask for guidance on pricing, features, or integration options.
